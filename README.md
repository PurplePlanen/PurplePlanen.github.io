## 🌈 介绍

这是一个使用 VitePress 构建的个人导航网站，用于收集和分享有趣的网站和工具。主题基于 [vitepress-nav-template](https://github.com/maomao1996/vitepress-nav-template)。

## ✨ 特性

- 🎨 简洁美观的界面设计
- 📱 响应式布局，支持移动端访问
- 🌓 支持深色/浅色主题切换
- 🔍 支持网站搜索
- 📊 支持访客统计
- 💬 支持评论功能
